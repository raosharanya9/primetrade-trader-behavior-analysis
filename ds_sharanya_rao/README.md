# Trader Behavior Analysis Under Market Sentiment

This repository contains my submission for the
**Junior Data Scientist – Trader Behavior Insights** role.

## Project Overview
The objective of this project is to analyze how trader performance and behavior
vary under different Bitcoin market sentiment regimes (Fear vs Greed).

The analysis combines:
- Historical trader-level data from Hyperliquid
- Bitcoin Fear & Greed Index
